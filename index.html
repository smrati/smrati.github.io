3 different ways to declare variables in dart

Published: Sat 01 January 2022
By Smrati Kumar Katiyar

In Mobile Development.

tags: Flutter dart

  1. var
  2. final
  3. const

we will learn about difference in these 3 types

mutable, static type

main() {
  mutableBasicType();
}

void mutableBasicType(){
  int a = 5;
  double b = 12.7;
  bool c = true;


  print(a);
  print(b);
  print(c);

}

A variable once assigned as int can't be reassigned as double

main() {
  mutableBasicType();
}

void mutableBasicType(){
  int a = 5;
  double b = 12.7;
  bool c = true;

  a = 45.6;


  print(a);
  print(b);
  print(c);

}

values of different type assigned to same variable as it was declared as dynamic

main() {
  mutableDynamic();
}

void mutableDynamic(){
  dynamic a = 14.6;
  print(a);
  a = false;
  print(a);
}

How to identify runtimetype

main() {
  mutableDynamic();
}

void mutableDynamic(){
  dynamic a = 14.6;
  print(a);
  print(a.runtimeType);
  a = false;
  print(a);
  print(a.runtimeType);
}

next we are going to understand what is var variable type once we declare a variable with var type it means when we assign some value to this varibale dart will set the type based on that value and once the type is infered based on the value supplied , that point onwards it can't be changed. so let's understand it with an example.
