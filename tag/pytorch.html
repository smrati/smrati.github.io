Highlevel Overall workflow to develop a deep learning model using PyTorch

Published: Thu 15 October 2020
By Smrati Kumar Katiyar

In pytorch.

tags: Pytorch

what is the usual flow of deep learning based model training

  1. You preprocess the data
  2. Define the model
  3. train the model in loop
  4. test model performance
  5. tweak hyperparameters and repeat step 3 to 5

If even after hyperparamter tuning we don't see improvement that mean you need to revisit your choice of model.

To give you an end to end example abut how to implement above 5 steps in pytorch, we will give you example code with CIFAR-10 dataset

Dataset -> accessing and preprocessing data from a file or data source
Sampler -> Sampling data from dataset to create batches
DataLoader -> iterate over batches
